class pupil_recording_interface.MotionReader(folder, stream='odometry', source='recording')¶

Parameters
folderstr or pathlib.Path

Path to the recording folder.

streamstr, default ‘odometry’

The motion stream to read. Can be ‘odometry’, ‘accel’ or ‘gyro’.

sourcestr, default ‘recording’

The source of the data. So far, only ‘recording’ is supported.
